🎯 **What:**
Split the overly large `src/utils/github/github.ts` into two specialized files:
- `api.ts` handles GitHub API operations (PRs, issues, listing).
- `git.ts` handles local Git commands (`runGit`, `gitFetch`, `gitPush`, etc.).
