## Description
This PR addresses a performance issue in the `commit-and-open-pr` tool where pre-commit reviewers were invoked sequentially inside a `for...of` loop. The execution of independent agent reviews has been parallelized using `Promise.all` with `.map()`, removing the O(N) waiting penalty for remote API calls and LLM invocation.
