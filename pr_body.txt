💡 **What:** Refactored the `listArtifacts` and `cleanupArtifacts` functions in `src/utils/memory-pointer.ts` to execute file reads and asynchronous operations concurrently using `Promise.all()` instead of an iterative `for...of` sequential loop.

🎯 **Why:** Reading files one by one using `await readFile` block execution and results in inefficient I/O operations and latency.

📊 **Impact:** Reading 5,000 files dropped from ~478ms to ~23ms (a ~20x performance improvement).

🔬 **Measurement:** Measured by reading the directory, simulating reading 5,000 files using sequential loops vs `Promise.all()` mapped over the array. Verified correctness against `bun test src/utils/__tests__/memory-pointer.test.ts`.
